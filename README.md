## 🚀 快速安装

### 基础依赖安装
```bash
# 必须首先安装原项目依赖
git clone https://github.com/Byaidu/PDFMathTranslate.git
cd PDFMathTranslate
pip install -r requirements.txt

# 然后安装本分支新增依赖
cd ..
git clone https://github.com/yourname/PDF-Translator.git
cd PDF-Translator
pip install -r requirements.txt
```

### 启动程序
```bash
python main.py
```
